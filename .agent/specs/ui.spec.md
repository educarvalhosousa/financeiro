# UI/UX Specification (.spec)

## Estética "Premium Glass"
O projeto deve manter um visual de alta gama, fugindo do básico.

## Pilares de Design
- **Fundo**: Dark mode profundo ou Glassmorphism sutil.
- **Cores Dinâmicas**:
  - Receitas: `#4ade80` (Green-400)
  - Despesas: `#f87171` (Red-400)
  - Acento: `#8b5cf6` (Violet-500)
- **Interação**: Micro-animações em botões e transições de modal.
- **Feedback**: Loading states não-bloqueantes e toasts informativos (StatusMessages).

## Componentes Críticos
- **Dashboard**: Layout em grades (Grid) responsivas.
- **FilterBar**: Acesso rápido a períodos e membros da casa.
- **TransactionList**: Itens compactos com identificação clara do autor (Membro).
